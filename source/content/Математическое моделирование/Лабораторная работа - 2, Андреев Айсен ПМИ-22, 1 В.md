## ex_07
### Problem
Построить график функции в декартовой системе координат с pm3d:
1. Эллипсоид
### Solution
```
#!/usr/bin/env gnuplot

set terminal png font "Arial,10"
set output "ex_07.png"

set xrange [-2:2]
set yrange [-2:2]
set zrange [-2:2]

set parametric
set isosamples 30, 30
set hidden3d

a = 1.5
b = 1.0
c = 0.8

set urange [0:pi]
set vrange [0:2*pi]

splot a*sin(u)*cos(v), b*sin(u)*sin(v), c*cos(u) with lines title "Эллипсоид"
```

## ex_08
### Problem
Построить график функции, заданной параметрически:
 ![[Pasted image 20260114122439.png]]
### Solution
```
#!/usr/bin/env gnuplot

set terminal png font "Arial,10"
set output "ex_08.png"

set title "Катеноид (Catenoid)"
set grid

set parametric

set isosamples 40, 40
set hidden3d

set urange [0:2*pi]
set vrange [-2:2]

c = 1.0

splot c*cosh(v/c)*cos(u), c*cosh(v/c)*sin(u), v with lines lc rgb "#2c3e50"
```
